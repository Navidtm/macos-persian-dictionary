# Screenshots

This folder will hold screenshots of the dictionary running inside
Dictionary.app on macOS (search results, the sidebar entry, the About page,
etc.).

To add one after building and installing the dictionary locally:

1. Open Dictionary.app and look up one of the 20 seed words (e.g. "patience").
2. Take a screenshot (`Cmd+Shift+4`, then `Space` to capture just the window).
3. Save it here as a descriptive PNG, e.g. `patience-entry.png`.
4. Reference it from the [Screenshots section of the README](../../README.md#screenshots).
