English -> Persian Dictionary for macOS
========================================

INSTALL
-------
1. Double-click "Install.command".
   - The first time, macOS will likely warn that it's from an
     "unidentified developer." Right-click (or Control-click) the file
     instead, choose "Open", then confirm "Open" in the dialog. You only
     need to do this once.
2. A Terminal window will appear briefly and Dictionary.app will open
   automatically once installation finishes.
3. In Dictionary.app, go to Dictionary > Settings (or Preferences) and
   enable "English-Persian" in the list of dictionaries.

That's it - no Python, Xcode, or command-line tools needed.

UNINSTALL
---------
Double-click "Uninstall.command" the same way.
