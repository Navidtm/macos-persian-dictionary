English → Persian Dictionary

An open-source English-to-Persian dictionary for macOS, built for the native Dictionary.app. Every entry is written by hand with full definitions, IPA pronunciation, example sentences, synonyms, antonyms, related words, and common idioms — not just a raw word-for-word translation.

About this edition

This is an early, intentionally small edition covering a curated set of core English vocabulary. The project is designed to grow into a much larger community-maintained dictionary over time.

Contributing

New words, corrections, and improved translations are welcome. See the project's README.md and CONTRIBUTING.md for how to add a new entry.
